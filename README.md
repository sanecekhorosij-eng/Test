# Space Station UI Prototype

Интерактивный прототип игрового приложения в научно-фантастическом стиле.

## Экраны

- Главная страница
- Настройки
- Колесо фортуны
- Сундук наград

## Навигация

На главном экране:
- ⚙️ открывает настройки
- 🎡 открывает колесо фортуны
- 🧰 открывает сундук

На внутренних экранах кнопка «Назад» возвращает на главную страницу.

## Запуск

Откройте `index.html` в браузере или запустите локальный HTTP-сервер из корня проекта.

Например:

```bash
python -m http.server 8000
```

После этого откройте `http://localhost:8000`.

## Структура

- `index.html` — разметка приложения
- `styles.css` — оформление и адаптивность
- `app.js` — навигация, таймеры и интерактивность
- `assets/` — изображения интерфейса

## Статус

Проект находится в активной разработке. Текущая версия — интерактивный UI-прототип.
