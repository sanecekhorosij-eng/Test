# Space Station UI Prototype

Интерактивный прототип игрового приложения в научно-фантастическом стиле.

## Экраны

- Главная страница
- Настройки
- Колесо фортуны
- Сундук наград

## Модульная структура

- `src/pages/home.js` — только главная страница
- `src/pages/chest.js` — только страница сундука
- `src/pages/wheel.js` — только страница колеса
- `src/pages/settings.js` — только страница настроек
- `src/pages/index.js` — подключение модулей страниц
- `src/config/page-backgrounds.js` — единый блок для замены фонов страниц
- `src/bootstrap.js` — сборка интерфейса перед запуском приложения
- `app.js` — текущая навигация, таймеры и действия
- `styles.css` — оформление и адаптивность
- `assets/` — изображения интерфейса

## Навигация

На главном экране:
- ⚙️ открывает настройки
- 🎡 открывает колесо фортуны
- 🧰 открывает сундук

На внутренних экранах кнопка «Назад» возвращает на главную страницу.

## Запуск

Из-за использования JavaScript-модулей проект нужно запускать через локальный HTTP-сервер из корня репозитория:

```bash
python -m http.server 8000
```

После этого откройте `http://localhost:8000`.
